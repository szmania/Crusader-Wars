# Linux/Proton Setup Wizard - Implementation Plan

## 1. Summary / Overview

### Problem Statement
The current Linux/Proton setup process for Crusader Wars requires 10+ manual steps including:
- Wine prefix creation and configuration
- .NET 4.7.2 installation via winetricks
- Mod symlinking for Attila
- Shortcut creation for Attila launcher
- Steam launch options configuration
- Proton compatibility settings

This complex multi-step process creates a high barrier to entry for Linux users.

### Solution
Create an in-app wizard that automates the entire setup process into a single-click operation, reducing user effort from 10+ manual steps to a guided wizard with minimal user input.

### Expected Outcomes
- ✅ One-click setup wizard accessible from the main application
- ✅ Automatic detection of Linux/Proton environment
- ✅ Automated Wine prefix setup with .NET 4.7.2
- ✅ Automatic mod symlinking for Attila
- ✅ Shortcut creation and Steam configuration
- ✅ User-friendly progress tracking and error handling
- ✅ Support for both CK3 and Attila path detection

## 2. Design Requirements

### 2.1 User Requirements
- **UR-001**: Wizard should be accessible from the main application UI (Options or dedicated button)
- **UR-002**: Clear step-by-step progress indication
- **UR-003**: Minimal user input required (only game paths if not auto-detected)
- **UR-004**: Real-time feedback on setup progress
- **UR-005**: Error messages with actionable solutions
- **UR-006**: Ability to re-run setup if needed

### 2.2 Software Requirements
- **SR-001**: Detect if running on Linux/Proton environment
- **SR-002**: Detect installed Wine version
- **SR-003**: Detect Steam installation and game paths
- **SR-004**: Execute shell commands via Wine/Proton
- **SR-005**: Create symlinks for Attila mods
- **SR-006**: Create Windows shortcuts (.lnk files)
- **SR-007**: Modify Steam launch options
- **SR-008**: Install .NET 4.7.2 via winetricks
- **SR-009**: Handle file system operations in Linux paths
- **SR-010**: Log all operations for debugging

### 2.3 Compatibility Requirements
- **CR-001**: Support for Steam and non-Steam game installations
- **CR-002**: Support for different desktop environments (GNOME, KDE, etc.)
- **CR-003**: Compatible with Proton 7.0+
- **CR-004**: Support for various Wine configurations
- **CR-005**: Handle different Linux distributions (Ubuntu, Fedora, Arch, etc.)

### 2.4 Performance Requirements
- **PR-001**: Setup should complete within 5-10 minutes (mostly .NET installation)
- **PR-002**: Non-blocking UI with progress updates
- **PR-003**: Minimal disk space usage for temporary files

## 8. Technical Notes

### 8.1 Path Translation
Wine uses Z:\ to represent the Linux root (/). The wizard must handle path conversions:
- Linux: `/home/user/Games/SteamGames/Total War Attila`
- Wine: `Z:\home\user\Games\SteamGames\Total War Attila`

### 8.2 Command Execution
All shell commands must be executed through Wine:
```bash
WINEPREFIX=~/.crusader-wars-net-pfx wine [command]
```

### 8.3 Symlink Considerations
- Symlinks should use absolute paths
- Check for existing files before creating symlinks
- Handle permission errors gracefully

### 8.4 Steam Launch Options
Example launch options for Attila:
```
PULSE_LATENCY_MSEC=60 PROTON_ENABLE_WAYLAND=1 mangohud gamemoderun %command% used_mods_cw.txt
```
